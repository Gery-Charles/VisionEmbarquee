# 👁️ Vision Embarquée

Travaux Pratiques de traitement d'images et vision embarquée sur caméra **JeVois**.  
Cours réalisé en école d'ingénieurs · **Python** · 6 TPs

---

## 📁 Structure du projet

```
VisionEmbarquee/
├── TP1/   # Prise en main de la caméra JeVois — Suivi par corrélation
├── TP2/   # Estimation de mouvement
├── TP3/   # Rétroprojection de l'histogramme — CamShift
├── TP4/   # Réalité augmentée — Aruco
├── TP5/   # Transformée de Hough — Télémètre
└── TP6/   # Détection de visages — Haar
```

## ⚙️ Prérequis

- Python 3.8+
- OpenCV, NumPy

```bash
pip install opencv-python numpy
```

Pour les TPs sur caméra JeVois, se référer à la [documentation officielle JeVois](http://jevois.org/doc/).
